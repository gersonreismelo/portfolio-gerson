Candidate Hiring System

Descrição: Este projeto foi dedicado ao desenvolvimento de um chatbot voltado para auxiliar o setor de Recursos Humanos (RH) na seleção de candidatos para vagas de emprego. A principal função do chatbot era realizar uma pré-entrevista com os candidatos, classificando e ranqueando os perfis mais adequados para a vaga.

Resultados: O chatbot implementado atendeu integralmente às necessidades do setor de RH, fornecendo uma solução eficiente para a triagem inicial de candidatos. Este projeto recebeu reconhecimento significativo, sendo selecionado e prestigiado pela IBM como um dos Top 10 projetos inovadores. Além disso, foi apresentado no prestigioso evento NEXT, demonstrando seu impacto e relevância.

Tecnologias Utilizadas:

Utilizamos metodologias de desenvolvimento como Diagramas e Product Backlog para planejamento e estruturação do projeto.

Na implementação, foram empregadas tecnologias como Java para a lógica de backend, REST para comunicação entre sistemas, React para a interface de usuário e Oracle para armazenamento de dados.

Contribuí como um dos desenvolvedores principais no projeto, participando ativamente de todas as fases do ciclo de desenvolvimento.

Este projeto foi uma experiência enriquecedora e demonstrou a capacidade de criar soluções inovadoras que se destacam no mercado, promovendo eficiência e eficácia nos processos de Recrutamento e Seleção.

SeniorSmart

Descrição: Este projeto teve como foco o desenvolvimento de um aplicativo destinado a auxiliar a população idosa com dificuldades no uso da tecnologia. O objetivo era criar um assistente virtual com inteligência artificial (IA) avançada para orientar passo a passo sobre o uso de tecnologias cotidianas, como configuração de celulares, realização de compras online, entre outros.

Resultados: O aplicativo desenvolvido foi concebido para ser altamente intuitivo e fácil de usar, oferecendo orientações passo a passo para usuários idosos enfrentando desafios tecnológicos. A integração de um assistente virtual baseado em IA permitiu oferecer suporte personalizado para dúvidas e orientações diversas. Este projeto recebeu destaque, sendo selecionado e prestigiado pela Plusoft no Top 12, além de ter sido apresentado no evento Next.

Tecnologias Utilizadas:

Utilizamos metodologias de desenvolvimento, como Diagramas e Product Backlog, para o planejamento e estruturação do projeto.

Na implementação, foram empregadas tecnologias como Java, Spring para o backend, Next.js e React Native para interfaces web e mobile, Oracle para armazenamento de dados e integração com a API do ChatGPT para aprimorar a interação com o assistente virtual. Aplicamos também a esteira devops na Azure.

Contribuí como um dos desenvolvedores principais no projeto, participando ativamente de todas as fases do ciclo de desenvolvimento.

Este projeto representa uma iniciativa inovadora que busca reduzir a lacuna tecnológica entre as gerações e fornecer suporte valioso para a população idosa, facilitando a adoção e uso de tecnologias modernas.
